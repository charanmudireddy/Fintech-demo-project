# Fintech-demo-project
Production grade cloud-native demo platform for a loan management API using AWS, Terraform, Docker, Kubernetes, Helm, ArgoCD, GitHub Actions, PostgreSQL, Prometheus, Grafana, ELK-style logging approach, Bash, and Python.

## Features

- Python API for loan records
- PostgreSQL backend
- Dockerized application
- Kubernetes deployment
- Helm chart
- GitHub Actions CI
- ArgoCD GitOps deployment
- Prometheus metrics
- Grafana dashboard integration
- Bash deployment and backup scripts
- Terraform AWS infrastructure

## Run locally
# Bash
docker compose up --build
